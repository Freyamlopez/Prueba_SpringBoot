📢 *Detalles de la prueba técnica*

⏱️ *Duración:* 3 horas.

Durante ese tiempo deberán desarrollar el proyecto de acuerdo con las especificaciones indicadas.

📌 *Entrega:*

* Subir el proyecto a una rama con el nombre: *SASF_NOMBRE_APELLIDO*.

* No utilizar herramientas de IA durante la prueba.

⭐ *Puntos extra:*

* Uso de DTOs.
* Mappers.
* Documentación con Swagger.
* Logs.
* Auditoría.
